Windows下搭建Hexo博客的基础步骤

工具：windows下：npm、Git；

建议有npm使用经验再来搭建博客，npm命令也十分简单，只是windows下安装配置稍显麻烦，按照网上博客步骤一步一步来就行，强烈建议设置淘宝源镜像，不要问为什么没有Linux教程，因为我现在不太会Linux，等用熟Linux之后我会回来更新的。

一、下载npm，换上国内淘宝源头cnpm

二、下载Hexo插件

npm install -g hexo-cli

验证hexo是否下载成功：

hexo -v

三、建个文件夹（存放博客文件）

四、在该文件夹下：

hexo init

该命令是初始化hexo博客文件夹，包括一些配置文件和一个主题

看到最后一句话证明hexo博客初始化完成；

五、关于操纵hexo博客的一些命令：

hexo s:启动博客程序；

hexo n “文章名”:写一篇文章；

hexo clean:清理一下博客文件，多使用这个命令就好；

hexo g:生成博客文件，使其能够被预览;

…还有很多命令，我也暂时不知道，知道了回过头来再更把

六、Github部署并获取域名访问

  • 在自己的GitHub上创建仓库名，仓库名必须为：Github用户名.github.io

  • 在本地安装如下插件：

    npm install –save hexo-deployer-git

  • 设置_config.yml:

    打开_config.yml文件，找到deploy属性添加如下内容

    deploy:

    ​ type: git

    ​ repo: github仓库地址

    ​ branch: master（该属性指明的应该是哪个分支）

  • 接下来仍然在博客文件夹下，输入

    hexo d:该命令意思是将本地博客文件部署到Github仓库，仓库地址就是在上述配置文件里面写的地址

  • 至此，博客部署完成，在浏览器地址栏输入Github用户名.github.io即可直接远程访问博客
